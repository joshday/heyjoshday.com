AdaGrad has a Learning Rate

AdaGrad (PDF) is an interesting algorithm for adaptive element-wise learning rates. First consider a stochastic gradient descent (SGD) update

θ(t)=θ(t−1)+γtgt, \theta^{(t)} = \theta^{(t-1)} + \gamma_t g_t, θ(t)=θ(t−1)+γt​gt​,

where γt\gamma_tγt​ is a step size (learning rate) and gt=∇ft(θ(t−1))g_t = \nabla f_t(\theta^{(t-1)})gt​=∇ft​(θ(t−1)) is the gradient of a noisy objective function. The AdaGrad algorithm replaces the step sizes γt\gamma_tγt​ with a matrix inverse that scales the elements of the gradient:

θ(t)=θ(t−1)+diag(Gt)−12gt, \theta^{(t)} = \theta^{(t-1)} + \text{diag}(G_t)^{-\frac{1}{2}}g_t, θ(t)=θ(t−1)+diag(Gt​)−21​gt​,

where Gt=∑i=1tgtgtTG_t = \sum_{i=1}^t g_t g_t^TGt​=∑i=1t​gt​gtT​. In this form, it is suggested that AdaGrad does not have a learning rate. However, with very basic algebra, the AdaGrad update is

θ(t)=θ(t−1)+1tdiag(Gt∗)−12gt, \theta^{(t)} = \theta^{(t-1)} + \frac{1}{\sqrt t}\text{diag}(G_t^*)^{-\frac{1}{2}}g_t, θ(t)=θ(t−1)+t​1​diag(Gt∗​)−21​gt​,

where Gt∗=t−1GtG_t^* = t^{-1} G_tGt∗​=t−1Gt​. That is, if the matrix term producing the adaptive step sizes is considered as a mean instead of a sum, we see that AdaGrad has a learning rate γt=1/t\gamma_t = 1/\sqrt tγt​=1/t​.
